## 專案介紹
運用react框架自製一款todo-list，使用者可以
* 查看自己的所有 todo
* 新增 todo、刪除 todo並標記完成/未完成的 todo，
* 雙擊清單上的特定 todo 編輯其內容，可以註冊 / 登入 / 登出

## How to use
* npm install 安裝所需要的檔案
* npm start 啟動專案
* 另外開啟一終端機，並運行 npm run dev-server
